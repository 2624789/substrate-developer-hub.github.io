介绍

In this tutorial, you will learn to compile and launch a pre-written Substrate chain, use that chain as a cryptocurrency, and interact with it using a general-purpose Substrate user interface.

本教材针对之前从未接触过Substrate、希望能简单、 快速地了解Substrate全貌的人。 我们不会过多地深入Substrate开发的内部细节，希望能满足你的好奇心，从而可以让你继续这段旅程。

学习本教程需要花费你1小时左右的时间。 We will be compiling rust code, but you do not need to be a rust programmer to complete this guide. We provide you with working code and explain how to use it.

我们只期望：

  • 您基本熟悉如何进行软件开发和使用终端。
  • 您基本熟悉区块链和智能合约平台。
  • 您希望了解区块链开发的前沿技术。

If you run into an issue on this tutorial, we are here to help! You can create a new issue or contact us on Riot.

您将做什么

Before we even get started, let's lay out what we are going to do over the course of this tutorial. 我们将：

  1. 设置您的计算机可以在 Substrate 上开发。
  2. 使用模板项目立即运行 Substrate
  3. Use a general-purpose user interface to interact with our blockchain.
  4. Transfer tokens between user accounts.

还不错吧？ 好，让我们开始吧！
