密码学

本文档提供了Substrate中使用的加密技术的概念概述。

哈希算法

Substrate使用哈希函数将任意大小的数据映射到固定大小的值上。

Substrate provides two hash algorithms out of the box, but can support any hash algorithm which implements the Hasher trait.

xxHash

xxHash是一种快速的非加密型哈希函数，算法速度接近于RAM的极限。 由于xxhash不是加密安全的，所以有可能通过修改输入来合理地控制散列算法的输出。 这允许使用者通过创建密钥碰撞、哈希碰撞和 imbalanced storage tries来攻击此哈希算法。

xxHash用于外部各方无法操纵哈希函数输入的地方。 例如，它用在生成runtime存储值的主键，其输入由runtime开发人员控制。

Substrate使用了Rust实现的 twox-hash 。

Blake2

Blake2 是一个基于密码学的哈希函数。 它是公认的非常快，也被用于 Zcash 中。

Substrate使用了Rust实现的blake2。

公钥加密

Substrate 使用公钥加密技术提供可靠的认证系统。

Substrate provides multiple different cryptographic schemes and is generic such that it can support anything which implements the Pair trait.

ECDSA

Substrate 提供一个基于secp256k1曲线的 ECDSA 签名方案。 这个加密算法也用于Bitcoin和 Ethereum。

Ed25519

Ed25519 是一个基于Curve25519曲线的EdDSA签名方案， 它在设计和实现上经过多个级别进行了精心设计，表现出极高的速度而又不影响安全性。

SR25519

SR25519 is based on the same underlying curve as Ed25519. 然而，它使用Schnorr签名替换了EdDSA的原有方案。

Schnorr 签名给 ECDSA/EdDSA 方案带来了一些显著的功能：

  • 对于分层确定性密钥派生而言更好用的表现。
  • It allows for native multi-signature through signature aggregation.
  • 它更能防止滥用。

在ECDSA上使用Schnorr签名时，做出的一个牺牲是两者都需要64个字节，但是只有ECDSA签名才可以传递其公钥。
