Compiling the Tools

Substrate (目前) 没有提供二进制安装包，因此必须从源代码中编译它，这可能是一个耗时的过程。 It is wise to start the compilation and then read the next few pages of the tutorial while your node is building. This tutorial will use the Substrate node template (version pre-2.0-3e65111).

# Install rust prerequisites
curl https://getsubstrate.io -sSf | bash -s -- --fast

# Download the Substrate node template
git clone -b pre-v2.0-3e65111 --depth 1 https://github.com/substrate-developer-hub/substrate-node-template

现在让我们编译我们将要运行的区块链节点。 由于 Substrate 是一个框架，大多数现实世界中的区块链将使用它编写自定义的 runtime 代码。 There are other tutorials that cover that process in detail. Luckily the node template is a ready-to-run node environment.

# Ensure your Rust toolchain is up to date
./scripts/init.sh

# Compile the node-template
cargo build --release
